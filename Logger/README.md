# Logger
Goal of this project is to deeply understand how professional logging libraries are designed and implemented by building every major components.

# Project Goals
Trying to Build a reusable logging library for any C++ application.
Learn modern C++ through a real-world systems project.
Design clean and extensible APIs.
Understand multithreading, synchronization, and asynchronous programming.
Explore high-performance systems programming techniques.
Create a library that can be reused in future personal and professional projects.
